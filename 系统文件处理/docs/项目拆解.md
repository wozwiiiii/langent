## 项目拆解
# 项目名称 -- 基于langchain的RAG系统文件处理系统（智盘）

## 1.核心需求
使用用户：
- 大学生 / 研究生
- 知识工作者 / 上班族
- 研究人员 / 自学者

解决问题：智能处理杂乱的系统文件，包括文件搜索、分类、整理、备份以及删除和恢复。

得到结果：能够帮助用户更高效地组织和管理文件（将不必要的文件删除），同时能够根据用户需求智能分类整理文件。


## 2.核心步骤拆解
1.文件搜索：
   - 用户输入搜索关键词（如"学习资料"、"项目文件"等）
   - 系统根据关键词及历史记录在本地文件系统中搜索相关文件（RAG基础策略，后期拓展为混合检索）
   - 返回搜索结果（文件列表）
   - 用户选择文件进行操作（如打开、删除、备份等）
   - 保存用户操作记录（如搜索关键词、操作类型、操作时间等）

2. 文件整理：
   - 用户选择文件夹对文件夹下的文件进行整理（如按时间、按大小类型等）
   - 系统根据用户选择的整理标准，将文件移动到对应的文件夹中
   - 用户可以查看整理结果（如文件夹下的文件列表）
   - 保存用户操作记录（如整理标准、整理时间等）

3. 文件备份：
   - 用户选择文件或文件夹进行备份（本地或云端）
   - 系统根据用户选择的备份方式，将文件备份到指定位置（本地或云端）
   - 用户可以查看备份结果（如备份文件路径）
   - 保存用户操作记录（如备份方式、备份时间等）

4. 文件删除：
   - 用户选择文件或文件夹进行删除
   - 系统根据用户选择的删除方式，将文件删除（本地或云端）
   - 用户可以查看删除结果（如删除文件路径）
   - 保存用户操作记录（如删除方式、删除时间等）
   - 同时可以依据用户操作记录，自动删除的文件或文件夹（需备份）
